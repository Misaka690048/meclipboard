# UI 设计规范

## 1. 配色方案（米白色主题）

| 用途 | 色值 | 色块 | 说明 |
|------|------|------|------|
| 页面背景 | `#FFF8F0` | ██ | 暖白/米色基调 |
| 卡片背景 | `#FAF0E6` | ██ | 略深的米色，与背景区分 |
| 卡片悬停 | `#F5E6D3` | ██ | 鼠标悬停时略微加深 |
| 卡片边框 | `#DEB887` | ██ | burlywood，温和的轮廓线 |
| 置顶标识 | `#CD853F` | ██ | peru 色，4px 左边框 |
| 主文字 | `#3E2723` | ██ | 深棕色，阅读舒适 |
| 次要文字 | `#8D6E63` | ██ | 中棕色，用于时间戳等 |
| 交互元素 | `#D2B48C` | ██ | tan 色，按钮和强调 |
| 删除按钮 | `#E57373` | ██ | 柔红，避免刺眼 |
| 搜索框背景 | `#FFFFFF` | ██ | 纯白，与米色背景区分 |

## 2. 布局尺寸

|------|-----|

### 2.2 搜索栏

| 属性 | 值 |
|------|-----|
| 位置 | 固定顶部 |

| 高度 | 48px |

| 圆角 | 8px |

| 属性 | 值 |

### 2.4 卡片

| 属性 | 值 |
|------|-----|

|------|-----|

### 2.6 设置面板

| 属性 | 值 |

## 3. 字体规范

| 用途 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 全局 | `system-ui, -apple-system, "Segoe UI", sans-serif` | 14px | normal |
| 卡片文字内容 | 继承 | 13px | normal |
| 卡片时间戳 | 继承 | 11px | 300 |
| 搜索输入框 | 继承 | 14px | normal |
| 面板标题 | 继承 | 16px | 600 |
| 按钮文字 | 继承 | 12px | 500 |

---

## 4. 图标

- 优先使用简单 Unicode 符号：📌（置顶）、✕（删除）、⚙（设置）、🔍（搜索）
- 如后续需要更精致的图标，使用内联 SVG

---

## 5. 交互反馈

| 操作 | 反馈 |
|------|------|
| 卡片悬停 | 背景色加深 + 阴影增强，过渡 150ms ease |
| 卡片点击（回贴） | 短暂蓝绿色闪烁 + 文字变为"已复制！"1.5 秒后恢复 |
| 删除卡片 | 卡片向右滑出 300ms + 底部出现"已删除 · 撤销"提示条 3 秒 |
| 置顶/取消置顶 | 按钮图标即时切换 |
| 搜索过滤 | 卡片淡入淡出 200ms |

---

## 6. 对话框 / 提示样式

- 撤销提示条：底部固定，深色背景（`#3E2723`），白色文字，带"撤销"链接
- 无模态对话框（避免打断操作），所有消息使用轻量级 Toast
