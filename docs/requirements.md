# 需求规格说明书

## 1. 项目概述

**项目名称**：ClipHistory（Windows 历史剪贴板应用）

**目标**：为 Windows 用户提供剪贴板历史管理功能，自动记录文字和图片复制内容，支持搜索、置顶、回贴等操作，界面简洁直观。

---

## 2. 功能需求

### 2.1 剪贴板监听

| 项目 | 描述 |
|------|------|
| **描述** | 应用在后台运行，自动监听并记录用户复制到系统剪贴板的所有内容 |
| **支持类型** | 文字（纯文本）和图片（位图） |
| **监听方式** | 每 500ms 轮询剪贴板，检测内容变化 |
| **去重策略** | 文字：直接字符串比较；图片：SHA-256 哈希值比较 |
| **验收标准** | 复制文字或截图后，应用窗口中自动出现对应记录卡片 |

### 2.2 历史浏览（卡片列表）

| 项目 | 描述 |
|------|------|
| **描述** | 以卡片形式展示所有剪贴板历史记录 |
| **排序** | 默认按时间降序（最新在上），置顶项始终在最前 |
| **文字卡片** | 显示文字内容预览（前 100 字符），超出省略号截断 |
| **图片卡片** | 显示缩略图（最大高度 120px） |
| **时间显示** | 相对时间（刚复制 / X 分钟前 / X 小时前 / 具体日期） |
| **验收标准** | 打开窗口可看到历史记录；置顶项排在最前；文字和图片均正确展示 |

### 2.3 搜索功能

| 项目 | 描述 |
|------|------|
| **描述** | 通过关键词快速查找历史记录 |
| **匹配方式** | 大小写不敏感的文本匹配 |
| **响应方式** | 实时过滤（输入时即时更新卡片列表，250ms 防抖） |
| **搜索范围** | 仅搜索文字内容（图片不可搜索） |
| **验收标准** | 输入关键词后卡片列表实时缩小显示匹配项；清空搜索框恢复全量列表 |

### 2.4 置顶功能

| 项目 | 描述 |
|------|------|
| **描述** | 将重要条目固定在列表顶部 |
| **操作** | 每张卡片右上角有置顶按钮，点击切换置顶/取消置顶 |
| **视觉标识** | 置顶卡片左侧有 4px 宽的深色边框（#CD853F）和 📌 图标 |
| **持久化** | 置顶状态存入数据库，重启应用后保持 |
| **验收标准** | 置顶后卡片移到最前，重启后仍保持；取消置顶后卡片回到按时间排序的位置 |

### 2.5 删除功能

| 项目 | 描述 |
|------|------|
| **描述** | 手动删除不需要的历史记录 |
| **操作** | 每张卡片右上角有删除（✕）按钮 |
| **撤销** | 删除后显示 3 秒"已删除，点击撤销"提示条 |
| **图片清理** | 删除图片类型记录时同时删除对应的磁盘文件 |
| **验收标准** | 点击删除后卡片消失；3 秒内可撤销恢复；图片文件一并清理 |

### 2.6 粘贴回剪贴板

| 项目 | 描述 |
|------|------|
| **描述** | 点击历史记录卡片，将对应内容写回系统剪贴板 |
| **反馈** | 点击后卡片短暂闪烁 + 显示"已复制"提示 |
| **文字回贴** | 文字内容写入系统剪贴板 |
| **图片回贴** | 从磁盘加载图片文件，写入系统剪贴板 |
| **验收标准** | 点击文字卡片后在记事本 Ctrl+V 粘贴成功；点击图片卡片后在画图 Ctrl+V 粘贴成功 |

### 2.7 存储配置

| 项目 | 描述 |
|------|------|
| **保留天数** | 用户可设置保留天数（默认 30 天），过期自动清理 |
| **容量上限** | 用户可设置最大占用空间（默认 500MB），超出后自动清理最旧的未置顶项 |
| **置顶保护** | 自动清理不删除置顶项 |
| **清理时机** | 应用启动时执行一次 + 每 60 分钟执行一次 |
| **验收标准** | 修改保留天数为 1 天，重启后旧记录被清理；填满至超容量后自动清理旧项 |

### 2.8 系统集成

| 项目 | 描述 |
|------|------|
| **开机自启** | 用户可在设置中开启/关闭开机自动启动 |
| **系统托盘** | 关闭窗口时隐藏到托盘，后台继续监听剪贴板 |
| **托盘菜单** | 右键图标显示"显示"和"退出" |
| **全局快捷键** | Alt+V 呼出/隐藏主窗口 |
| **验收标准** | 关机重启后应用自动运行；关闭窗口后 Alt+V 可唤出；托盘右键菜单功能正常 |

---

## 3. 非功能需求

| 项目 | 描述 |
|------|------|
| **隐私** | 所有数据存储在本地，不连接网络，不上传任何数据 |
| **性能** | 后台运行时内存占用 < 60MB |
| **窗口大小** | 默认 400×600px，最小 300×400px |
| **平台** | Windows 10/11 |

---

## 4. 数据存储路径

```
C:\Users\{用户名}\AppData\Local\com.clipboard-app.app\
├── clipboard.db          # SQLite 数据库
└── images\               # 图片文件
    └── {hash}.png
```
